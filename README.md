# ML-course-project
CS 1675 - Intro to Machine Learning - Course Project

Cleaned Dataset: https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset

The Kaggle dataset was constructed from the raw BRFSS 2015 survey data by Alex Teboul. His data preparation notebook (linked below) documents the feature selection and cleaning steps. We extended his pipeline to include race/ethnicity (`_RACE`) sourced directly from the CDC's raw XPT file. The BRFSS 2015 codebook provides full documentation for all survey variables.

Teboul data preparation notebook: https://www.kaggle.com/code/alexteboul/diabetes-health-indicators-dataset-notebook#Diabetes-Health-Indicators-Dataset-Notebook

CDC BRFSS 2015 data: https://www.cdc.gov/brfss/annual_data/annual_2015.html

CDC BRFSS 2015 codebook: https://www.cdc.gov/brfss/annual_data/2015/pdf/codebook15_llcp.pdf
